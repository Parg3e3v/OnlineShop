
# ТЗ на разработку приложения для онлайн-магазина

## Стек технологий
- Kotlin 
- Jetpack Compose
- Корутины
- Flow 
- Dagger Hilt
- MVVM
- Clean Architecture
- Многомодульность 
- Room
- Retrofit
- Proto DataStore
